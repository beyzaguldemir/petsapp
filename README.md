# 🐾 PetsApp — Sanal Evcil Hayvan Oyunu

## 📖 Proje Açıklaması

PetsApp, kullanıcının sanal bir evcil hayvanla (Buddy) ilgilenebildiği mobil bir uygulamadır. Buddy'yi besleyebilir, oynatabilir, uyutabilir ve mini oyunlar oynayarak coin kazanabilirsiniz. Uygulama React Native ve Expo ile geliştirilmiştir.

---

## ✨ Özellikler

- 🍗 **Besleme** — Buddy'nin açlık seviyesini yönet
- 😊 **Oynatma** — Aktiviteler ve mini oyunlarla mutluluğunu artır
- 🛁 **Banyo** — Buddy'yi temiz tut
- 😴 **Uyku Modu** — Buddy uyurken tüm etkileşimler kilitlenir
- 🛒 **Mağaza** — Kazanılan coinlerle ürün satın al
- 📊 **Seviye Sistemi** — Biriken coinlerle seviye atla

---

## 🎮 Mini Oyunlar

- **🦘 Jump Game** — Platform platformdan zıplayarak yukarı tırman. Sol/sağ butonlarla veya klavye ok tuşlarıyla hareket et.
- **🍬 Match Game** — 8×8 grid üzerinde aynı renkteki pet'leri 3'lü eşleştir. 60 saniye süren Candy Crush tarzı oyun.
- **🐟 Catch Game** — Yukarıdan düşen yiyecekleri sürükleyerek pet'in ağzıyla yakala.

Her oyun sonu coin ve mutluluk puanı kazanılır.

## 🚀 Projeyi Çalıştırma

### Gereksinimler
- Node.js (v18+)
- Expo Go uygulaması (Android) — SDK 54 destekli
- Bilgisayar ve telefon aynı Wi-Fi ağında olmalı

### Komutlar

```bash
# Bağımlılıkları yükle
npm install --legacy-peer-deps

# Uygulamayı başlat
npx expo start -c
```

Terminalde QR kod çıktıktan sonra **Expo Go** uygulamasından tarat.
